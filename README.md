# 疾风小恐龙：像素风跑酷游戏

疾风小恐龙是一款基于经典 Chrome 小恐龙游戏（Dinosaur Game）的像素风格跑酷游戏。本项目旨在复刻原版游戏的核心玩法，并在此基础上进行功能扩展与优化，为玩家提供更丰富的游戏体验。

---

## 游戏特色

- **复古像素风格**：致敬经典的 Chrome 小恐龙游戏，还原像素艺术的魅力。
- **现代技术实现**：使用 C++ 和 Qt 框架开发，确保高性能与跨平台支持。
- **动态难度系统**：游戏速度随时间或分数逐渐加快，挑战玩家反应极限。
- **昼夜交替与彩蛋**：新增昼夜切换、生日彩蛋和奥运彩蛋等趣味功能。
- **音效与交互**：为跳跃、碰撞、得分等事件添加音效，提升沉浸感。

## 开发

项目结构：
- src/: C++ 源代码和资源文件
- include/: 公共头文件
- assets/: 图片和其他二进制资源
- build/: 推荐的外部构建目录

快速构建（zsh）：

```zsh
mkdir -p build
cd build
cmake .. -DCMAKE_BUILD_TYPE=Debug
cmake --build .
```

如果 CMake 找不到 Qt，请传递前缀路径，例如：
```zsh
cmake .. -DCMAKE_PREFIX_PATH="/path/to/Qt/6.6.2/clang_64"
```
